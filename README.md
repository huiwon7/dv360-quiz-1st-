# Google DV360 퀴즈 게임

Google Display & Video 360(DV360) 초보자와 중급자를 위한 인터랙티브 퀴즈 게임입니다.

## 레벨 구성

### Level 1 - 초보자 퀴즈
- DV360의 기본 개념과 용어
- 총 10문제
- 파란색/보라색 테마

### Level 2 - 중급자 퀴즈
- 심화된 DV360 지식
- 총 15문제
- 핑크색 테마
- 문제 랜덤 섞기 기능

## 기능

- 반응형 디자인 (PC, 태블릿, 모바일 지원)
- 실시간 점수 표시
- 정답/오답 즉시 피드백
- 애니메이션 효과
- 최종 결과 및 평가 메시지
- 레벨 간 이동 기능

## 사용 방법

메인 페이지에서 원하는 레벨을 선택하여 퀴즈를 시작하세요!
